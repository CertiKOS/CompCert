This directory contains the source code of CompCertELF,
an extension of CompCert that supports verified separate
compilation of C programs to relocatable ELF files.

Consult the following CompCertELF paper for details:

   CompCertELF: Verified Separate Compilation of C Programs into ELF Object Files

The structure of the source code is as follows.
